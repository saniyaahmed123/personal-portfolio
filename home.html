Task Management App

A full-featured task management web app developed using React and Firebase. It enables users to create, update, and organize daily tasks with real-time synchronization and authentication. I designed the UI with accessibility and productivity in mind — offering an intuitive workflow, color-coded priorities, and live status tracking to enhance team collaboration.

E-commerce Store
